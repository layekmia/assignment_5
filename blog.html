Question-1: What are the different ways to select an element in the DOM?

JavaScript has many ways to select an element. For example: getElementById(): select a single element using id attribute, getElementsByClassName: This return a collection of html, getElementByTagName() Select all lements with specific tag name, querySelector(): Selects the first element that matches a CSS selector, querySelectorAll(), This return a nodeList

Question-3: What is event delegation in the DOM?

Event delegation is assigning an event listener to a parent element to handle events for child elements. It's efficient for dynamically added elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event propagates from the target element up to the root.
